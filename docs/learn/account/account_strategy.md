# 계좌 구성 전략 — 어떤 계좌 먼저 채울까

## 한 줄로

> 세제 혜택 큰 순서대로 — **연저펀 → ISA → IRP → 직투**. DC형은 회사가 알아서.

## 5가지 계좌 비교

| 계좌 | 한도 (연) | 세제 혜택 | 인출 제약 | 추천 용도 |
|---|---|---|---|---|
| **연금저축펀드 (연저펀)** | 600만 | 세액공제 13.2~16.5% + 과세이연 | 55세 이전 인출 = 페널티 큼 | **최우선**. 노후 자금 |
| **IRP (개인형 퇴직연금)** | 900만 (연저펀 합산) | 동일 + 추가 한도 | 55세 페널티 동일 | 연저펀 다 채운 후 추가 300만 |
| **ISA (개인종합자산관리)** | 2,000만 (5년 누적 1억) | 비과세 200만~400만 + 초과 9.9% 분리과세 | 의무 가입 3년 (이전 인출 시 혜택 없음) | 단기/중기 (3~5년) 자금 |
| **DC형 퇴직연금** | 회사 납입 | 운용수익 과세이연 → 55세 후 연금소득세 (3.3~5.5%) | 퇴직 시까지 묶임 | 회사가 자동. 운용만 선택 |
| **일반 (직투)** | 무제한 | 국내주식 비과세 / 해외 양도세 22% (250만 공제) | 없음 | 즉시 인출 자금 |

## 우선순위 — 이렇게 채워라

### Step 1: DC형 (회사가 자동)

회사가 매월 알아서 납입. 너는 **운용만 선택**:
- 안전 ETF 위주 (TIGER 200, KODEX 미국S&P500 등)
- 위험자산 비중 70% 한도 (퇴직연금 규정)
- 한도 차감 X — **별도 한도** 라서 부담 없음

### Step 2: 연저펀 600만 (세액공제 최대)

세액공제율:
- **총급여 5,500만 이하** → 16.5% (= 600만 납입 시 99만원 환급)
- **총급여 5,500만 초과** → 13.2% (= 79.2만원 환급)

ETF 자유롭게 매매 가능. 한국에 상장된 ETF만 (해외주식 직접투자는 X).

### Step 3: ISA 2,000만 (단기/중기 자금)

- 만기 3년 → 그 안에 인출하면 비과세 혜택 못 받음
- 만기 도래 시 연저펀/IRP 로 이체하면 추가 한도 300만 + 세액공제 받음 (꿀팁)

### Step 4: IRP 추가 300만 (연저펀 한도 다 채웠을 때)

연저펀 + IRP 합산 900만 한도. 600만 연저펀 채웠으면 IRP 300만 더 가능.

### Step 5: 직투 (남는 돈)

- 국내주식 매도 차익 비과세 (대주주 제외)
- 해외주식 매도 차익 22% (250만원 공제)
- 즉시 인출 가능 — 비상금/단기 트레이딩

## 케이스: 다중 IRP / DC 보유

> **개인 IRP (삼성증권) + 회사 DC (농협) + 미래 신한 DC (이직 시)** — 가능한가?

가능. 정리:

| 시점 | DC | 개인 IRP | 메모 |
|---|---|---|---|
| 현 직장 (농협 DC) | 농협 활성 | 삼성증권 활성 | 동시 운용 OK |
| 1년 휴직 / 무직 | 농협 DC 그대로 유지 | 삼성 IRP 계속 납입 가능 | 휴직 중에도 IRP 납입 OK |
| 신규 직장 (신한 DC) | 농협 → **신한 IRP로 이체 권장** (혹은 신한 DC 신설 후 이체) | 삼성 IRP 그대로 | 농협 DC 그대로 두면 운용 신경 안 쓰임 |

**핵심**: DC 형은 회사별로 별도 계좌. 이직 시 → IRP 로 이체하거나, 새 회사 DC 로 합치거나. **개인 IRP 와는 무관** (별도 운용).

## 주의사항

- **연저펀 ≠ 연금저축보험**: 보험 형태는 ETF 매매 안 됨. **펀드 / ETF 형 (연저펀)** 이 트레이딩 자유로움
- 55세 이전 인출 시 **기타소득세 16.5%** 부과 — 세액공제분 다 토해냄 + 추가
- ISA 의무가입 3년은 **첫 입금일 기준**, 마지막 입금일 아님
- DC 와 DB 형 구분 — **DB (확정급여)** 는 회사가 운용. 본인이 운용하는 건 **DC (확정기여)**

## 실용 결론

1. 신입~사회초년: DC + 연저펀 (월 50만)
2. 결혼/내집마련 단계: ISA 추가
3. 안정기: IRP 추가 + 직투 분산
