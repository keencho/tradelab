# 해외주식 양도소득세

## 한 줄로

> 해외주식 매도 차익에 **22% 분리과세** (양도세 20% + 지방세 2%). 연 250만원 공제.

국내주식은 대주주 아니면 비과세지만, 해외주식은 무조건 세금.

## 계산 공식

```
과세 = (총 매도차익 − 총 매도손실 − 250만원) × 22%
```

## 예시

| 상황 | 매도차익 | 매도손실 | 과세 대상 | 세금 |
|---|---|---|---|---|
| 케이스 1 | 500만 | 0 | 250만 | 55만 |
| 케이스 2 | 500만 | 300만 | 0 (250 한도 내) | 0 |
| 케이스 3 | 1,000만 | 200만 | 550만 | 121만 |
| 케이스 4 | 200만 | 0 | 0 (공제 한도 내) | 0 |

## 손익통산 (Tax-Loss Harvesting)

같은 해 안에서 **익절 + 손절을 같이** 하면 세금 최적화.

### 시나리오

12월 현재:
- NVDA: +600만 평가익
- TSLA: -200만 평가손

**선택 A**: NVDA만 매도 → 과세 (600-250)×22% = 77만
**선택 B**: NVDA + TSLA 동시 매도 → 과세 (600-200-250)×22% = 33만

→ 손절 후 **즉시 같은 종목 재매수** 가능 (한국은 wash-sale rule 없음 = 미국과 다름).

## 신고

- **매년 5월** 종합소득세 신고 기간에 양도세 신고
- 증권사가 양도소득세 신고 대행 자료 제공 — 그걸 홈택스에서 옮기면 됨
- 250만원 이내면 신고만 하고 세금 X (자진신고 안 하면 가산세)

## 함정

| 함정 | 설명 |
|---|---|
| FX 환차익 포함 | 매수 시 환율 1300, 매도 시 1400 이면 환차익도 양도세 대상. 원화 기준 차익 |
| 250만 = 인당 한도 | 부부 합산이 아님. 각자 250만 |
| 연저펀/IRP 안 적용 | 이 계좌들 안에서 해외 ETF 매매는 **과세이연** (양도세 X, 인출 시 연금소득세) |
| 매도 시점 기준 | 평가익은 세금 X. **매도해서 차익 확정**된 것만 |

## 실용 팁

- **12월 말 손절 검토** 루틴화 → 손익통산 적용
- 큰 익절 예정이면 같은 해에 손실 종목 같이 정리
- 한국주식 매매로는 양도세 발생 안 함 (대주주 제외) — 별개로 봐라
- 250만 한도 잘 활용: 매년 일정 부분 익절하면 효율적
