# 외국인 / 기관 순매수

## 한 줄로

> 하루 동안 **외국인 (또는 기관) 이 산 금액 − 판 금액**. 양수면 순매수, 음수면 순매도.

KRX 가 매일 장 마감 후 공시. pykrx 로 받아옴.

## 왜 보는가

한국 주식시장에서 **외국인 지분율 ~35%**, 기관 ~20%. 개인 ~45%. 큰돈이 어디로 가는지 보는 게 합리적.

- **외인이 며칠 연속 매수** → 추세 형성 가능성
- **외인 + 기관 동반 매수** → 흔치 않음. 신호 강함
- **외인 매도 + 개인 매수** → 개인이 받아주는 상황. 보통 약세 신호

## TradeLab 에서

`foreign_net_buy` 시그널 — z-score 가 2.5 이상이면 (= 평소 거래 패턴 대비 이례적) 알림. 단순히 "오늘 외인이 1000억 샀음" 보다 **그 종목 평소 외인 거래량 대비 얼마나 이례적인지** 보는 게 핵심.

## 함정

| 함정 | 설명 |
|---|---|
| 외인 ≠ 스마트머니 | "외국인" 안에는 헤지펀드, 인덱스 패시브, 차익거래, 단기 트레이딩 다 섞여있음. 패시브는 그냥 지수 추종일 뿐 |
| 거래소-외-외인 | 다크풀/블록딜은 즉시 안 잡힘. 다음 날 정정될 수 있음 |
| 프로그램 매매 | 차익거래 프로그램은 선물-현물 차이 잡으려고 일시적으로 산다/판다. 방향성 의미 약함 |

## 실용 팁

- **수급은 가격을 만든다**. 다만 단기 (1~3일) 관점에서. 장기 추세는 펀더멘털
- 외인 매수 + 외인 보유 비중 증가 동시 확인이 강함 (단순 1일 매수만 보면 위험)
- 코스피200 편입 종목은 외인 영향이 훨씬 큼. 중소형주는 기관이 더 중요
