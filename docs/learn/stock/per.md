# PER (Price-to-Earnings Ratio, 주가수익비율)

## 한 줄로

> **현재 주가 ÷ 주당순이익(EPS)**. "이 회사 1주를 사면, 회사가 1년에 버는 돈의 몇 배를 내고 사는가?"

PER 10 이면 회사가 지금처럼 벌면 10년 만에 원금 회수.

## 왜 보는가

같은 업종 안에서 **싼지 비싼지** 가늠하는 가장 기본 지표. 삼성전자 PER 12 vs SK하이닉스 PER 18 → 같은 반도체인데 하이닉스가 더 비싸게 거래 중이라는 뜻.

## 함정

| 함정 | 설명 |
|---|---|
| 업종 무시하고 비교 | 반도체 PER 12 와 헬스케어 PER 40 비교는 의미 없음. **업종 평균과 비교해야 함** |
| 일회성 이익 | 작년에 자산 매각 같은 일회성 이익이 EPS 부풀린 경우 PER 가 비정상적으로 낮게 보임 |
| 적자 회사 | EPS 가 마이너스면 PER 자체가 의미 없음 (음수, 또는 N/A) |
| 미래 무시 | 성장주는 PER 50 이라도 합리적일 수 있음 (앞으로 이익이 클 거니까). 반대로 사양산업 PER 5 도 비싼 거일 수 있음 |

## Forward PER vs Trailing PER

- **Trailing PER (TTM)**: 지난 12개월 실제 이익 기준 → 사실 기반, 보수적
- **Forward PER**: 향후 12개월 추정 이익 기준 → 미래 반영, 그러나 추정은 빗나감

뉴스나 토스 앱에 그냥 "PER" 이라고 쓰여있으면 보통 **Trailing**.

## 실용 팁

- 단독으로 보지 말고 **PBR/ROE 함께** 봐라
- 분기 실적 발표 직후 PER 가 갑자기 떨어졌으면 → EPS 가 올랐다는 것 (좋은 신호일 수 있음)
- 시장 전체 PER (KOSPI 평균 PER) 와 비교하면 **시장 대비** 싼지 가늠 가능
