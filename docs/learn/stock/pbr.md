# PBR (Price-to-Book Ratio, 주가순자산비율)

## 한 줄로

> **주가 ÷ 주당순자산**. "회사를 지금 청산하면 주주가 받을 돈 대비, 시장은 몇 배를 내고 있는가?"

PBR 1 = 장부가 그대로. PBR 0.5 = 절반 가격. PBR 3 = 장부가의 3배.

## 왜 보는가

- **PBR < 1** 은 "회사 자산 다 팔아도 시가총액보다 큼" → 이론적으로 저평가
- 한국 시장은 PBR 1 미만 종목이 흔함 (지배구조 디스카운트, "코리아 디스카운트")

## 함정

| 함정 | 설명 |
|---|---|
| 자산이 진짜 그 가치인가 | 장부상 부동산이 실제 시세보다 비싸게 잡혀있을 수 있음. 반대로 IT 회사의 무형자산은 장부에 안 잡힘 |
| 이익이 안 나면 의미 없음 | PBR 0.5 인데 매년 적자면, 자산이 계속 까이는 중 → 진짜 싼 게 아님 |
| 사양산업 | 조선/철강처럼 사양산업은 만년 PBR 1 미만. 싸다고 사면 영원히 거기 있음 |

## PER과 함께 봐야 하는 이유

| 케이스 | PER | PBR | 해석 |
|---|---|---|---|
| 가치주 가능성 | 낮음 | 낮음 | 이익도 나고 자산도 두꺼움. 1차 후보 |
| 성장주 | 높음 | 높음 | 미래 기대치 반영 (위험도 큼) |
| 자산주 | 적자 | 낮음 | 회사는 적자지만 자산이 두꺼움 |
| 함정 | 낮음 | 낮음 + 계속 그대로 | 시장이 외면하는 이유가 있음 |

## 실용 팁

- 정부의 "**밸류업 프로그램**"은 PBR 1 미만 회사를 타겟. 정책 모멘텀 받을 수 있음 (2024~ 한국)
- 금융주(은행/보험)는 PBR 위주로 본다 (이익 변동성이 크기 때문)
