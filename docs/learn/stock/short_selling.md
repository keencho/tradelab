# 공매도 (Short Selling)

## 한 줄로

> **주식을 빌려서 먼저 판 후, 가격이 떨어지면 싸게 사서 갚는** 거래. 떨어져야 돈을 번다.

## 어떻게 작동하나

1. 증권사나 기관에서 주식 100주 빌림 (수수료 지급)
2. 시장에 100주 매도 (현재가 10,000원 → 100만원 확보)
3. 가격이 떨어지면 (예: 8,000원) 100주 매수 (80만원)
4. 빌린 주식 반납 → 차익 20만원 수익

## 왜 보는가

**공매도 잔고/비중**: 해당 종목에 누가 떨어진다고 베팅하는 규모
- 잔고가 높음 → 약세 신호 (스마트머니가 떨어진다고 봄)
- **잔고가 급격히 증가** → 공격적인 숏 베팅, 더 강한 약세 신호

## TradeLab 에서

`short_ratio` 시그널 — 거래일 대비 공매도 거래 비중이 z-score 2.5+ 면 알림.

## 양면성

| 관점 | 해석 |
|---|---|
| 약세 신호 | 누군가 하락 베팅 중 |
| 강세 가능성 (반전) | "**쇼트 스퀴즈**" — 공매도 비중이 너무 높아지면, 가격이 오를 때 숏들이 손실 최소화 위해 강제로 매수해야 함 → 가격 폭등 |

GameStop (2021) 이 전형적 쇼트 스퀴즈 사례.

## 한국 시장 특수성

- 2023.11~2025.3: **공매도 전면 금지** (정치적 이슈) — 시그널 자체가 무력화됨
- 2025.4부터 재개. 하지만 룰 변경 (잔고/벌금 강화)
- 외국인 + 기관만 공매도 가능. 개인은 사실상 X (대주거래 제한적)

## 실용 팁

- 공매도 비중만 보고 사고팔지 말 것. **가격 + 거래량 + 수급 같이** 봐라
- 공매도 잔고는 지연 공시 (T+2). 실시간 아님
- 코스피200/코스닥150 만 공매도 가능. 일반 중소형주는 무관
