# z-score 시그널

## 한 줄로

> 어떤 값이 **과거 평균에서 표준편차 몇 배 떨어져 있는지**. 절대값이 크면 이례적.

## 공식

```
z = (현재값 − 과거 평균) / 과거 표준편차
```

## 해석

| |z| | 해석 | TradeLab 처리 |
|---|---|---|
| < 2 | 정상 범위 | 시그널 생성 X |
| 2 ~ 2.5 | 약간 이례적 | 화면 표시만 |
| 2.5 ~ 3 | 이례적 | **알림 발송** |
| 3+ | 매우 이례적 | **알림 발송** (강한 신호) |

## 왜 절대값을 안 쓰나

"외인이 1,000억 순매수했다" 만 보면 종목 크기에 따라 의미가 다름.
- 삼성전자에서 1,000억 → 평소 수준
- 중소형주에서 1,000억 → 매우 이례적

z-score 는 **각 종목의 평소 패턴 대비 얼마나 튀는가** 를 보기 때문에 종목 크기와 무관하게 비교 가능.

## TradeLab 에서 사용하는 곳

- `foreign_net_buy` — 외인 순매수
- `institutional_net_buy` — 기관 순매수
- `short_ratio` — 공매도 비중
- `funding_rate` — 코인 펀딩
- `open_interest` — 코인 OI
- `naver_buzz` — 검색 빈도
- `reddit_buzz` — Reddit 언급
- `whale_transfer` — 고래 이체

## 함정

| 함정 | 설명 |
|---|---|
| 과거 데이터 부족 | 새 종목/이벤트 직후엔 평균/표준편차가 불안정. z-score 가 과도하게 튐 |
| 추세 무시 | 꾸준히 상승 중인 지표면 z-score 가 계속 +. "평소" 자체가 변하는 중 |
| 정규분포 가정 | 펀딩이나 OI 는 비대칭 분포. z-score 만으로 100% 정확한 임계는 아님 |

## 쿨다운

같은 시그널이 너무 자주 오는 걸 막기 위해 **z-score 시그널은 12시간 쿨다운**.
- 외인이 3일 연속 이례적 매수 → 첫날만 알림, 그 후 12시간씩 무음
- 방향이 바뀌면 (+ → −) **30분 후 즉시 재알림 가능**

## 실용 팁

- z-score 단독 매매 X. **가격 + 다른 시그널과 조합**해서 판단
- z = 2.5 와 3.0 은 의미 다름. 절대값 큰 쪽이 강함
- 시그널 페이지에서 "평소 대비 N배" 라벨이 z-score 의 사람-친화적 표현
