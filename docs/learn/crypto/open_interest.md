# OI (Open Interest, 미결제약정)

## 한 줄로

> 현재 시장에 열려있는 **선물 계약의 총 명목가치**. 청산되거나 만기되지 않은 포지션 합계.

## 왜 보는가

OI 는 **거래량과 다름**. 거래량은 "오늘 얼마나 사고팔았나", OI 는 "지금 열려있는 베팅이 얼마인가".

| 가격 | OI | 해석 |
|---|---|---|
| 상승 + OI 증가 | 신규 롱 진입 → **추세 강함** |
| 상승 + OI 감소 | 숏 청산만 일어남 → 추세 약함 |
| 하락 + OI 증가 | 신규 숏 진입 → **하락 추세 강함** |
| 하락 + OI 감소 | 롱 청산만 일어남 → 추세 끝나가는 중 |

## TradeLab 에서

`open_interest` 시그널 — 평소 OI 대비 z-score 2.5+ 면 알림. **방향 + OI 변화 조합** 으로 판단.

## 실용 팁

- OI 가 빠르게 늘면 → **청산 후보 늘어남**. 변동성 커질 가능성 (위 아래 모두)
- OI 가 빠르게 줄면 → 시장 참여자 이탈, **추세 끝나가는 중**
- 펀딩레이트 + OI + 가격 → 코인 시장 3대 지표. 셋이 동조하면 강한 신호

## 단위

- 명목가치: USD (또는 USDT)
- 코인 수량: 약정한 코인의 총량 (BTC, ETH 등)

거래소마다 표기 방식 다름. TradeLab 은 보통 USD 명목가치로 본다.
