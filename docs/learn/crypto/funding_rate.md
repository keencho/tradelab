# 펀딩레이트 (Funding Rate)

## 한 줄로

> 코인 무기한 선물에서 **롱이 숏에게** (또는 그 반대로) 8시간마다 주는 수수료율. 시장이 한쪽으로 쏠려있을수록 커진다.

## 왜 존재하나

무기한 선물은 만기가 없음 → 현물 가격과 괴리될 수 있음. 펀딩레이트로 가격을 현물에 묶어둠.

- **롱 포지션이 많음** → 선물 가격 > 현물 → 롱이 숏에게 펀딩 지급 → 롱 비용 증가, 청산 유도 → 균형
- **숏이 많음** → 선물 < 현물 → 숏이 롱에게 지급

## 왜 보는가

**과열 신호** — 펀딩레이트가 비정상적으로 높음 (예: +0.1% per 8h = 연 +109%) 은 **너무 많은 사람이 롱**이라는 뜻. 시장 천장 근처에서 자주 나옴.

반대로 펀딩이 마이너스로 깊게 빠지면 → 숏 과열 → 바닥 가능성.

## TradeLab 에서

`funding_rate` 시그널 — Binance/Bybit 의 펀딩이 z-score 2.5+ 면 알림 (코인별).

## 정상 범위

| 펀딩레이트 (8h) | 연환산 | 해석 |
|---|---|---|
| 0.01% | ~10.95% | 정상 (롱 약간 우세) |
| 0.05% | ~54.75% | 과열 시작 |
| 0.1%+ | ~109%+ | 매우 과열 — 조정 임박 가능 |
| -0.05% | -54.75% | 숏 과열 — 반등 가능 |

## 함정

- 펀딩 단독으로 매매 X. **가격 + OI + 청산 데이터 같이** 봐라
- 펀딩 ≠ 직접적인 가격 신호. **선행 지표가 아니라 동조 지표**에 가까움
- 거래소별 펀딩이 다름 (Binance vs Bybit). 평균 또는 가중치
