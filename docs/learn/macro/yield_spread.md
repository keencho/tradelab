# 장단기 금리차 (Yield Spread, 10Y-2Y)

## 한 줄로

> **10년 만기 국채 금리 − 2년 만기 국채 금리**. 일반적으로 +. 마이너스가 되면 "역전".

## 정상 vs 역전

| 상태 | 의미 |
|---|---|
| **+0.5% ~ +2%** (정상) | 장기 빌려쓰는 게 단기보다 비쌈 → 미래 성장 기대 |
| **0 근처** (평탄) | 시장이 미래 성장에 회의적 |
| **마이너스 (역전)** | "**장단기 금리 역전**" — 시장이 가까운 미래에 경기침체 → 금리 인하를 예상 |

## 왜 침체 선행지표인가

장기 금리는 **미래 경제** 기대 반영. 단기 금리는 **현재 연준 정책** 따라감.

- 연준이 인플레 잡으려고 단기금리 올림 → 2Y 상승
- 시장은 그 때문에 향후 경기 둔화 + 금리 인하 예상 → 10Y 하락
- 결과: 10Y < 2Y → 역전

**역사적으로**: 1970년 이후 미국 침체 8번 중 7번이 역전 후 6~24개월 뒤 발생.

## 함정

| 함정 | 설명 |
|---|---|
| 타이밍 부정확 | 역전 후 침체까지 6개월~2년. 알람 났다고 즉시 주식 팔면 손해 |
| 역전 해소가 신호 | **역전이 풀릴 때** (다시 플러스 되는 순간) 가 실제 침체 시작에 가까움 |
| Fed 개입으로 왜곡 | QE/QT 가 장기 금리를 인위적으로 누름 → 시그널 약화 가능 |

## 실용 팁

- 역전 자체보다 **역전 → 정상 복귀** 시점을 더 주의 깊게 보라
- 10Y-3M 스프레드도 보조로 보라 (Fed 가 선호하는 지표)
- 채권/주식 비중 조절 트리거로 활용 (역전 → 단기채/현금 비중 ↑)
