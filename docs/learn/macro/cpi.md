# CPI (Consumer Price Index, 소비자물가지수)

## 한 줄로

> 가계가 사는 **소비재/서비스 바구니의 가격 변화율**. 미국 BLS 가 매월 발표.

## 왜 중요한가

- **연준 (Fed) 의 정책 목표** 가 물가 안정 (2% 인플레이션)
- CPI 가 예상보다 높음 → 금리 인하 늦춰짐 → 주식/채권 하락
- CPI 가 예상보다 낮음 → 금리 인하 가능 → 주식/채권 상승

발표일에는 **장 변동성이 극심**.

## 변수들

| 항목 | 설명 |
|---|---|
| **Headline CPI** | 전체 항목 포함 (식품+에너지 포함). 헤드라인용 |
| **Core CPI** | 식품+에너지 제외 — Fed 가 더 중요시. 변동성 낮은 추세를 봄 |
| **YoY (전년동월비)** | 1년 전 같은 달과 비교. 보통 이걸 봄 |
| **MoM (전월비)** | 직전 달 대비. 단기 추세 |

## 발표 일정

- **매월 둘째 주 화요일** (정확하진 않음) 미국 동부 시간 8:30am
- 한국 시간으로는 **밤 9시 30분 (서머타임 시 10시 30분)**
- 발표 직후 미국 주식 선물 + 비트코인 즉시 반응

## TradeLab 에서

`us_cpi` 시그널로 추적. 예상치 대비 ±0.1%p 이상 어긋나면 알림.

## 한국 CPI 와의 차이

- 한국은 매월 첫째 주 발표 (통계청)
- 한국 CPI 는 **한국은행 기준금리** 결정에 직접 영향
- 미국 CPI 가 더 시장 영향력 큼 (글로벌 자금 흐름 때문)

## 실용 팁

- 발표 직전엔 **포지션 잡지 마라**. 코인은 특히 위/아래 즉시 5% 휘둘림
- 발표 후 30분 안에 결정 짓는 게 좋음. 그 이후엔 추세 형성됨
- **Truflation** 같은 실시간 인플레 지표를 보조로 봐라 (BLS 데이터는 후행)
